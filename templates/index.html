Adams–Moulton Predictor–Corrector Calculator

Main focus: use the calculator below to solve first-order ordinary differential equations using an Adams–Bashforth predictor and Adams–Moulton corrector.

Mathematical Discussion

The Adams–Moulton Predictor–Corrector method is used to approximate the solution of an initial value problem:

$$y' = f(x,y), \quad y(x_0)=y_0$$

Because the Adams–Moulton formula is implicit, a predictor value is first computed using the Adams–Bashforth method. That predicted value is then substituted into the Adams–Moulton formula to obtain a corrected value.

Predictor Formula

$$y_{n+1}^{p}=y_n+\frac{h}{2}(3f_n-f_{n-1})$$

Corrector Formula

$$y_{n+1}=y_n+\frac{h}{12}(5f_{n+1}^{p}+8f_n-f_{n-1})$$

The first additional starting value is generated using RK4 so that the two-step predictor-corrector formula can begin.

Worked Example 1

Problem: Use the method to approximate the solution of:

$$y'=x+y,\quad y(0)=1,\quad h=0.1$$

Step 1: Compute the starting values

At $x_0=0$, $y_0=1$:

$$f_0=f(0,1)=0+1=1$$

Using a one-step RK4 starter gives:

$$y_1 \approx 1.1103417,\quad x_1=0.1$$

$$f_1=f(0.1,1.1103417)=0.1+1.1103417=1.2103417$$

Step 2: Predictor

$$y_2^p=y_1+\frac{h}{2}(3f_1-f_0)$$ $$y_2^p=1.1103417+\frac{0.1}{2}[3(1.2103417)-1]$$ $$y_2^p=1.1103417+0.05(2.6310251)$$ $$y_2^p=1.2418930$$

Step 3: Corrector

$$f_2^p=f(0.2,1.2418930)=0.2+1.2418930=1.4418930$$ $$y_2=y_1+\frac{0.1}{12}[5(1.4418930)+8(1.2103417)-1]$$ $$y_2=1.1103417+0.0083333(15.8921986)$$ $$y_2\approx1.2427767$$

Worked Example 2

Problem: Use the method to approximate the solution of:

$$y'=y-x^2+1,\quad y(0)=0.5,\quad h=0.2$$

Step 1: Compute the starting values

$$f_0=f(0,0.5)=0.5-0^2+1=1.5$$

Using a one-step RK4 starter:

$$y_1\approx0.8292933,\quad x_1=0.2$$

$$f_1=f(0.2,0.8292933)=0.8292933-0.2^2+1=1.7892933$$

Step 2: Predictor

$$y_2^p=y_1+\frac{h}{2}(3f_1-f_0)$$ $$y_2^p=0.8292933+\frac{0.2}{2}[3(1.7892933)-1.5]$$ $$y_2^p=0.8292933+0.1(3.8678799)$$ $$y_2^p=1.2160813$$

Step 3: Corrector

$$f_2^p=f(0.4,1.2160813)=1.2160813-0.4^2+1=2.0560813$$ $$y_2=y_1+\frac{0.2}{12}[5(2.0560813)+8(1.7892933)-1.5]$$ $$y_2=0.8292933+0.0166667(23.0947539)$$ $$y_2\approx1.2142059$$
