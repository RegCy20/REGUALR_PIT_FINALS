# Adams–Moulton Predictor–Corrector PIT

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open:

```text
http://127.0.0.1:5000
```

## Function Input Examples

- `x + y`
- `y - x**2 + 1`
- `sin(x) + y`
- `exp(x) - y`

Use `**` for powers. Example: `x**2`, not `x^2`.
